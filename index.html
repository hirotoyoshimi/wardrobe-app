接続設定

GitHub の fine-grained PAT（対象リポジトリのみ・Contents: Read and write）を 貼り付けてください。トークンとデータはこの端末内（localStorage）にのみ保存され、 シェルやリポジトリには一切送信・コミットされません。

fine-grained PAT の作り方

  • GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  • Repository access: Only select repositories → 対象リポジトリ（既定 hirotoyoshimi/wardrobe）
  • Permissions → Repository permissions → Contents: Read and write（Metadata: Read は自動付与）
  • 生成された github_pat_… を上に貼り付け → 接続テスト
  • 失効させたい時は同画面の token から Revoke
